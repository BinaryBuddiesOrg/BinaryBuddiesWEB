🚀 Binary Buddies Web

Dynamic Content Management for Your Website

v1.0.0 Odoo 18 Public API

📋 Overview

This module provides a comprehensive content management system for the Binary Buddies website, including models and public REST API endpoints for seamless frontend integration.

✨ Features

🎯 Featured Projects

Showcase your best work with project portfolios, results, and technology tags

👥 Team Members

Display your talented team with profiles, certifications, and social links

📝 Blog Posts

Share insights with categorized blog posts and featured articles

💼 Career Listings

Post job openings with requirements and responsibilities

🔌 API Endpoints

All endpoints are publicly accessible and return JSON data:

Featured Projects

GET /api/bbweb/projects
GET /api/bbweb/projects/<id>

Team Members

GET /api/bbweb/team
GET /api/bbweb/team/<id>

Blog Posts

GET /api/bbweb/blogs
GET /api/bbweb/blogs/featured
GET /api/bbweb/blogs?category=AI/ML
GET /api/bbweb/blogs/<id>

Career Listings

GET /api/bbweb/careers
GET /api/bbweb/careers?department=Engineering
GET /api/bbweb/careers/<id>

🎨 Key Capabilities

  • ✅ Full CRUD operations via Odoo UI
  • ✅ Image management with binary storage
  • ✅ Drag-and-drop sequence ordering
  • ✅ Category and tag management
  • ✅ Public REST API with CORS support
  • ✅ Active/Archived status management
  • ✅ Rich text editor for blog content
  • ✅ Relational data (tags, results, certifications)

🚀 Getting Started

  1. Install the module from the Apps menu
  2. Navigate to "Binary Buddies Web" in the main menu
  3. Start adding your content (projects, team, blogs, careers)
  4. Access the data via public API endpoints
  5. Integrate with your frontend application
